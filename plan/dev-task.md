## **Frontend**

- **Фреймворк**: Vue 3
- **UI-библиотеки**:
  - Quasar (готовые компоненты)
  - Chart.js (графики)
- **State management**: Pinia
- **PWA**:
  - Workbox (кэширование)
  - Service Worker (оффлайн-режим)
- **Роутинг**: Vue Router
- **Адаптивный дизайн**: CSS Grid/Flex + Mobile-first

## **Backend & Data**

- **База данных**: Firebase (Firestore + Auth)
- **Хранение файлов**: Firebase Storage

## **Testing**

- **Framework**: Jest
